About this Page:

This page offers the following AJAX and non-AJAX functionality:

AJAX Functionality (For AJAX Film retrieval, see the console for Text, JSON and XML formatting evidence):

  • A button to add a list of film records to the Google Cloud Database (using the 'createAllFilms' servlet).
  • Buttons to retrieve all film records from the server in Text, JSON or XML format, and place them in a HTML table (using the 'getAllFilms' servlet).
  • Forms to search for an individual film record by name, retrieve it from the server in Text, JSON or XML format, and place it in a HTML table (using the 'getFilm' servlet).
  • Form to insert a new film record into the database, and display confirmation in the browser (using the 'insertFilm' servlet).
  • Form to delete a film record from the database, and display a confirmation message in the browser window (using the 'deleteFilm' servlet).
  • Buttons to retrieve 'The Shawshank Redemption' film record from the server in Text, JSON or XML format, and place it in a HTML table (using the 'getFilm' servlet).

Non - AJAX Functionality:

  • Buttons to retrieve all film records in Text, JSON and XML formats (using the 'getAllFilms' servlet).
  • Forms to search for and retrieve individual film records in Text, JSON and XML formats (using the 'getFilm' servlet).
